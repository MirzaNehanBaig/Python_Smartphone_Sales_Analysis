# 📱 Python Smartphone Sales Analysis

## 📌 Project Overview
An exploratory data analysis (EDA) project focused on uncovering trends, pricing strategies, and consumer preferences in the smartphone market. This project utilizes Python to clean, manipulate, and visualize smartphone sales data, offering data-driven insights into market performance.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib / Seaborn

## 📊 Key Analysis & Insights
* **Pricing Strategy:** Analyzed the distribution of pricing tiers across different brands to identify premium vs. budget market segments.
* **Feature Impact:** Investigated how specific specifications (e.g., RAM, Storage, Battery Capacity) correlate with the retail price.
* **Top Performers:** Identified the highest-rated and most frequently purchased smartphone models within the dataset.
